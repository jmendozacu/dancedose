
Module: Domain Logo
Author: Remya <https://www.drupal.org/u/remyarose>


Description
===========
This module will provide uploading different logos for your subdomains

Requirements
============

* Domain access module


Installation
============
Download and copy the module in to module directory.


Configuration / Usage
=====
Configuration for uploading logo for each domain:
      admin/config/domain/domain_logo

This will update logos in each subdomains created through domain access module
