INTRODUCTION
------------

The Domain Traversal module offers easy navigation between domains for
logged in users.

 * For a full description of the module, visit the project page:
   https://drupal.org/project/domain_traversal

 * To submit bug reports and feature suggestions, or to track changes:
   https://drupal.org/project/issues/domain_traversal


REQUIREMENTS
------------

This module requires the following modules:

 * Domain Access (https://drupal.org/project/domain)


INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module. Visit:
   https://www.drupal.org/docs/8/extending-drupal-8/installing-drupal-8-modules
   for further information.


CONFIGURATION
-------------

 * Configure user permissions in Administration » People » Permissions
   - Traverse between assigned domains

   Users with this permission will be able to quickly traverse between the roles
   selected in their profile.

   - Traverse between all domains

   Users with this permission will be able to quickly traverse between all
   active domains.

 * Note that anonymous users cannot be given these permissions.
